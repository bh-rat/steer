# steer docs

The documentation site for steer, built with [Fumadocs](https://fumadocs.dev)
on Next.js and deployed to Vercel.

Content lives in `content/docs/` as MDX; sidebar order and grouping live in
the `meta.json` files next to it.

```bash
pnpm install
pnpm dev      # http://localhost:3000
pnpm build
```
